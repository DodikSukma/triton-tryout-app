Database Schema

Skema 4 database PostgreSQL — satu per microservice.

ℹ️ Setiap service punya database sendiri. Tidak ada FK constraint antar-database — konsistensi lintas service dijaga di application layer. UUID digunakan sebagai primary key di semua tabel.
