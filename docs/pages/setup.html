Setup & Instalasi

Cara menjalankan Triton Denpasar di mesin lokal.

📦
Prerequisites
ToolVersiKeterangan
Node.js≥ 18.xRuntime untuk semua service dan frontend
PostgreSQL≥ 14.xDatabase (bisa via Docker)
MakeanyUntuk menjalankan make dev / make stop
Docker (opsional)anyUntuk PostgreSQL + Redis via docker compose
🚀
Langkah Setup
1

Clone repository

git clone https://github.com/your-org/tritonapp.git
cd tritonapp
2

Jalankan PostgreSQL (via Docker)

docker compose up -d postgres redis

Atau gunakan instalasi PostgreSQL lokal di port 5432.

3

Buat 4 database

docker exec -it triton-postgres psql -U triton_user -c "CREATE DATABASE db_auth;"
docker exec -it triton-postgres psql -U triton_user -c "CREATE DATABASE db_user;"
docker exec -it triton-postgres psql -U triton_user -c "CREATE DATABASE db_soal;"
docker exec -it triton-postgres psql -U triton_user -c "CREATE DATABASE db_jawaban;"
4

Jalankan schema migration

docker exec -i triton-postgres psql -U triton_user -d db_auth    < services/auth-service/src/db/schema.sql
docker exec -i triton-postgres psql -U triton_user -d db_user    < services/user-service/src/db/schema.sql
docker exec -i triton-postgres psql -U triton_user -d db_soal    < services/soal-service/src/db/schema.sql
docker exec -i triton-postgres psql -U triton_user -d db_jawaban < services/jawaban-service/src/db/schema.sql
5

Seed data awal

cd scripts
npm install
npm run seed

Membuat akun default untuk testing (lihat tabel di bawah).

6

Install dependencies semua service

cd services/auth-service    && npm install && cd ../..
cd services/user-service    && npm install && cd ../..
cd services/soal-service    && npm install && cd ../..
cd services/jawaban-service && npm install && cd ../..
cd services/api-gateway     && npm install && cd ../..
cd frontend                 && npm install && cd ..
7

Jalankan semua service (development)

make dev       # jalankan semua service dengan ts-node-dev (hot-reload)

Atau untuk berhenti:

make stop      # hentikan semua service
⚠️ Logs service disimpan ke file, bukan ke terminal. Cek di logs/ per service jika ada masalah.
8

Jalankan frontend

cd frontend
npm run dev

Frontend tersedia di http://localhost:3000, API Gateway di http://localhost:4000.
