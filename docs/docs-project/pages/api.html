API Endpoints

Semua endpoint yang tersedia via API Gateway (port 4000). Klik endpoint untuk lihat detail body dan response.

ℹ️ Base URL: http://localhost:4000. Semua request yang memerlukan auth harus menyertakan cookie triton.sid secara otomatis (browser) atau via withCredentials: true (axios).
AUTH SERVICE — /auth/*
POST
/auth/login
Login user, buat session
PUBLIC›
Request Body
{ "email": "admin@triton.id", "password": "admin123" }
Response 200
{ "success": true, "data": { "userId": "uuid", "role": "admin", "email": "admin@triton.id" } }
// Cookie Set-Cookie: triton.sid=... (HttpOnly)
Response 401
{ "success": false, "error": "Email atau password salah" }
POST
/auth/logout
Hapus session, clear cookie
Auth›
Response 200
{ "success": true, "data": null, "message": "Berhasil logout" }
GET
/auth/me
Cek session aktif pengguna
Auth›
Response 200
{ "success": true, "data": { "userId": "uuid", "role": "guru", "email": "guru1@triton.id" } }
Response 401
{ "success": false, "error": "Unauthorized" }
POST
/auth/change-password
Ganti password pengguna yang login
Auth›
Request Body
{ "oldPassword": "lama123", "newPassword": "baru456" }
Response 200
{ "success": true, "data": null, "message": "Password berhasil diubah" }
USER SERVICE — /users/*
GET
/users?role=guru|siswa
Daftar semua user (dengan filter role opsional)
Admin›
Query Params
?role=guru   // filter hanya guru
?role=siswa  // filter hanya siswa
(tanpa role) // semua pengguna
Response 200
{ "success": true, "data": [
  { "id": "uuid", "email": "...", "role": "guru", "is_active": true, "profile": { "nama_lengkap": "..." } }
]}
POST
/users
Buat user baru (auth + profile)
Admin›
Request Body
{ "email": "siswa@triton.id", "password": "pass123", "role": "siswa",
  "nama_lengkap": "Budi Santoso", "kelas": "XII IPA 1",
  "no_telepon": "08123456789" }
Response 201
{ "success": true, "data": { "id": "uuid", "email": "...", "role": "siswa", "profile": {...} } }
Response 409 (email sudah ada)
{ "success": false, "error": "Email sudah terdaftar.", "code": "EMAIL_EXISTS" }
PATCH
/users/:id/active
Toggle aktif/nonaktif pengguna
Admin›
Request Body
{ "is_active": false }
Response 200
{ "success": true, "data": { "id": "uuid", "is_active": false } }

JAWABAN SERVICE — /sesi/*, /hasil/*, /riwayat
POST
/sesi
Mulai atau lanjutkan sesi tryout
Siswa›
Request Body
{ "tryout_id": "uuid" }
Response 201 (baru)
{ "success": true, "data": { "id": "uuid", "status": "berlangsung", "mulai_at": "2024-..." } }
Response 409 (sudah selesai)
{ "success": false, "error": "Tryout sudah dikerjakan.", "code": "ALREADY_COMPLETED" }
POST
/sesi/:sesiId/jawab
Simpan/update satu jawaban (upsert auto-save)
Siswa›
Request Body
// Soal PG:
{ "soal_id": "uuid", "opsi_id": "uuid" }

// Soal Essay:
{ "soal_id": "uuid", "jawaban_teks": "Jawabannya adalah..." }
Response 200
{ "success": true, "data": { "id": "uuid", "sesi_id": "uuid", "soal_id": "uuid", ... } }
POST
/sesi/:sesiId/selesai
Kumpulkan ujian + hitung nilai otomatis
Siswa›
Proses Internal
1. Fetch soal dari soal-service (role=admin agar is_benar tidak disembunyikan)
2. Ambil jawaban siswa dari db
3. Hitung: nilai = (Σ bobot soal benar / Σ total bobot) × 100
4. Simpan ke tabel `hasil`
Response 200
{ "success": true, "data": { "id": "uuid", "nilai": "87.50",
  "total_benar": 35, "total_soal": 40 } }
GET
/hasil/rekap/:tryoutId
Rekap semua siswa untuk satu tryout (guru/admin)
Guru/Admin›
Response 200
{ "success": true, "data": {
  "summary": { "total_peserta": 25, "rata_rata_nilai": 76.4, "nilai_tertinggi": 95.0, "nilai_terendah": 42.5 },
  "hasil": [{ "siswa_id": "uuid", "nama_siswa": "Budi", "kelas": "XII IPA 1",
    "nilai": 95.0, "total_benar": 38, "total_soal": 40,
    "selesai_at": "2024-...", "durasi_menit": 67 }] } }
GET
/hasil/:sesiId
