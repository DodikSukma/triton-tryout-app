# Task TRN-30: Image & Asset Generation for Landing Page

## Overview
This task documents the prompt design, specifications, and target file outputs for all landing page images and illustrations. The goal is to establish a premium, cohesive, and modern look for Bimbel Triton Denpasar using high-quality flat vector and realistic illustrations.

## 📂 Target Files & Impact Areas
All assets must be saved in the `landingpage/public/` directory:
- `/hero.png` (Hero banner illustration)
- `/program-sd.png`, `/program-smp.png`, `/program-sma.png` (Jenjang program illustrations)
- `/method-1.png`, `/method-2.png`, `/method-3.png` (Method description illustrations)
- `/alumni-1.png`, `/alumni-2.png`, `/alumni-3.png` (Alumni student avatars)
- `/gallery-1.png` to `/gallery-6.png` (Suasana belajar gallery grid)
